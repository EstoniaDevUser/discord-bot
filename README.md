# Discord Bot

Deploy to Railway:

1. Loo GitHub repo ja lae siia üles need failid.
2. Mine [Railway](https://railway.app/) → New Project → Deploy from GitHub.
3. Vali see repo.
4. Railway lisab Node.js automaatselt, käivita käsuga `node index.js`.
5. Lisa Environment Variable:
   - KEY: TOKEN
   - VALUE: sinu Discord bot token
6. Vajuta deploy → bot töötab 24/7
